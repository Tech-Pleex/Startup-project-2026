# Roadmap

This roadmap tracks future ideas separately from the current setup-wizard release work.
Concrete items can later become GitHub issues or feature branches when they are ready to build.

## Future Features

- Explore dashboard layout prototypes with more room for the main setup checklist.
- Consider moving the computer choice to the landing page for a clearer first decision.
- Add links to SketchUp video tutorials when the tutorial set is ready.
- Plan a separate dashboard refresh for student and teacher use cases.
- Package the Windows setup as a single executable or installer to reduce download friction for students and teachers.

## Later

- Improve Mac setup delivery after the Windows flow has been tested in class.
- Add clearer guidance for downloaded setup files if browser or Windows security prompts confuse users.

## Done

- Add GitHub Pages landing page.
- Add Windows setup wizard.
- Add local GF2 IT Dashboard.
- Add Windows setup ZIP packaging.
- Add dashboard and setup delivery checks.
