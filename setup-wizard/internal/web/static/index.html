Windows S-mode er aktiveret

Assistenten kan ikke fortsætte, mens computeren kører i S-mode. Kontakt din lærer eller IT, slå S-mode fra, og prøv igen.

Assistenten beder aldrig om adgangskoder, MitID eller UNI-Login. Elever indtaster kun oplysninger på officielle sider og i Windows' egne indstillinger.
